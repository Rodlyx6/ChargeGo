# ChargeGo 共享充电桩平台 - 产品原型与UI设计

## 1. 设计理念

### 1.1 设计原则
- **简洁直观**：界面简洁，操作流程清晰
- **响应式设计**：适配桌面端和移动端
- **视觉一致性**：统一的色彩、字体、图标风格
- **用户友好**：清晰的视觉反馈和操作提示

### 1.2 色彩方案
- **主色调**：蓝色系（#3B82F6）- 代表科技、信任
- **辅助色**：绿色系（#10B981）- 代表环保、充电
- **警告色**：橙色系（#F59E0B）- 代表提醒、倒计时
- **错误色**：红色系（#EF4444）- 代表错误、取消
- **背景色**：渐变背景（蓝色到绿色）- 营造科技感

### 1.3 字体方案
- **标题字体**：粗体、大号字体
- **正文字体**：常规字体、适中大小
- **数字字体**：等宽字体、清晰易读

## 4. UI 组件设计

### 4.1 按钮组件
- **主按钮**：蓝色渐变背景，白色文字，圆角
- **次按钮**：灰色背景，深色文字，圆角
- **危险按钮**：红色渐变背景，白色文字，圆角
- **禁用按钮**：灰色背景，半透明，不可点击

### 4.2 卡片组件
- **白色背景**：阴影效果，圆角
- **渐变背景**：蓝色到绿色渐变，用于突出显示

### 4.3 状态标签
- **空闲**：绿色背景，白色文字
- **预约中**：黄色背景，深色文字
- **充电中**：蓝色背景，白色文字
- **故障**：红色背景，白色文字
- **已完成**：蓝色背景，白色文字
- **已取消**：红色背景，白色文字

### 4.4 表单组件
- **输入框**：白色背景，灰色边框，圆角
- **选择框**：白色背景，蓝色边框（选中），圆角
- **下拉框**：白色背景，灰色边框，圆角

## 5. 交互设计

### 5.1 页面跳转
- **平滑过渡**：页面切换时使用淡入淡出效果
- **面包屑导航**：显示当前位置，方便返回

### 5.2 操作反馈
- **加载提示**：显示加载动画
- **成功提示**：绿色提示框，自动消失
- **错误提示**：红色提示框，需要手动关闭
- **确认对话框**：重要操作前弹出确认对话框

### 5.3 动画效果
- **按钮点击**：缩放效果
- **卡片悬停**：阴影加深，轻微上移
- **数字变化**：数字滚动动画
- **倒计时**：脉冲动画

## 6. 响应式设计

### 6.1 桌面端（≥1024px）
- 多列布局
- 侧边栏导航
- 大号字体和图标

### 6.2 平板端（768px - 1023px）
- 两列布局
- 顶部导航
- 中号字体和图标

### 6.3 移动端（<768px）
- 单列布局
- 底部导航
- 小号字体和图标
